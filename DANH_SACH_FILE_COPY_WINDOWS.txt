PHƯƠNG ÁN 1: CHẠY TRỰC TIẾP VỚI PYTHON (CẦN CÀI PYTHON)
───────────────────────────────────────────────────────────────

Các file BẮT BUỘC phải copy:
───────────────────────────────────────────────────────────────
✓ kiem_kho_app.py              - File chính của ứng dụng
✓ DuLieuDauVao.xlsx            - File dữ liệu Excel (BẮT BUỘC)
✓ CHAY_TREN_WINDOWS.bat        - Script tự động cài đặt và chạy
✓ requirements.txt             - Danh sách thư viện cần thiết

Cách sử dụng:
1. Copy tất cả 4 file trên vào cùng một thư mục trên Windows
2. Double-click vào CHAY_TREN_WINDOWS.bat
3. Script sẽ tự động cài đặt thư viện và chạy ứng dụng


PHƯƠNG ÁN 2: TẠO FILE .EXE (CHẠY KHÔNG CẦN PYTHON)
───────────────────────────────────────────────────────────────

BƯỚC 1: BUILD FILE .EXE (Trên máy Windows có Python)
───────────────────────────────────────────────────────────────

Các file cần copy để BUILD file .exe:
───────────────────────────────────────────────────────────────
✓ kiem_kho_app.py              - File chính của ứng dụng
✓ DuLieuDauVao.xlsx            - File dữ liệu Excel (BẮT BUỘC)
✓ requirements.txt             - Danh sách thư viện cần thiết
✓ SUA_LOI_BUILD.bat            - Script build file .exe (KHUYẾN NGHỊ)
   HOẶC
✓ TAO_FILE_EXE_WINDOWS.bat     - Script build file .exe
   HOẶC
✓ build_windows.bat            - Script build file .exe

Cách sử dụng:
1. Copy tất cả các file trên vào cùng một thư mục trên máy Windows CÓ PYTHON
2. Double-click vào SUA_LOI_BUILD.bat (hoặc TAO_FILE_EXE_WINDOWS.bat)
3. Đợi quá trình build hoàn tất (có thể mất vài phút)
4. File .exe sẽ nằm trong thư mục: dist\KiemKhoApp.exe

BƯỚC 2: COPY FILE .EXE SANG MÁY WINDOWS KHÁC (Để chạy)
───────────────────────────────────────────────────────────────

Sau khi build xong, copy 2 file sau sang máy Windows KHÁC:
───────────────────────────────────────────────────────────────
✓ dist\KiemKhoApp.exe          - File thực thi (đã build sẵn)
✓ dist\DuLieuDauVao.xlsx       - File dữ liệu Excel

Cách sử dụng:
1. Copy cả 2 file trên vào cùng một thư mục trên máy Windows khác
2. Double-click vào KiemKhoApp.exe để chạy
3. KHÔNG CẦN cài Python hay bất kỳ thư viện nào!

───────────────────────────────────────────────────────────────

Các file cần copy (sau khi đã build xong):
───────────────────────────────────────────────────────────────
✓ dist\KiemKhoApp.exe          - File thực thi (đã build sẵn)
✓ dist\DuLieuDauVao.xlsx       - File dữ liệu Excel

Cách sử dụng:

TÓM TẮT NHANH
───────────────────────────────────────────────────────────────

Để chạy lại trên Windows, bạn có 3 lựa chọn:

1. CHẠY TRỰC TIẾP (cần Python):
   → Copy: kiem_kho_app.py, DuLieuDauVao.xlsx, 
           CHAY_TREN_WINDOWS.bat, requirements.txt
   → Chạy: CHAY_TREN_WINDOWS.bat

2. BUILD FILE .EXE (cần Python để build):
   → Copy: kiem_kho_app.py, DuLieuDauVao.xlsx, 
           requirements.txt, SUA_LOI_BUILD.bat
   → Chạy: SUA_LOI_BUILD.bat
   → Kết quả: dist\KiemKhoApp.exe

3. CHẠY FILE .EXE (không cần Python):
   → Copy: dist\KiemKhoApp.exe, dist\DuLieuDauVao.xlsx
   → Chạy: Double-click KiemKhoApp.exe


FILE TỐI THIỂU CẦN THIẾT
───────────────────────────────────────────────────────────────

Nếu chỉ muốn copy tối thiểu để chạy trực tiếp:
✓ kiem_kho_app.py
✓ DuLieuDauVao.xlsx
✓ CHAY_TREN_WINDOWS.bat
✓ requirements.txt

(4 file này là đủ để chạy ứng dụng trên Windows có Python)
